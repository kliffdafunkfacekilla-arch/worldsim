# Assets Directory

This directory contains static assets for the application, including:

*   **`sprites/`**: Character and object sprites.
*   **`tiles/`**: Terrain and map tiles.
*   **`standalone_generator_v2.py`**: Script for generating or processing assets independently.
*   **`standalone_output/`**: Output directory for generated assets.

Assets are used by the Kivy client interfaces (`main_client.py` and `quad_nested_client.py`) for rendering the visual representation of the simulation.
